Creates an item definition, which includes all the information on a specific item type that is the same for all instances of that item

Parameters
tilethe tile number of the tile used to render that item
namethe item's name
descriptionthe item's description
equipSlotwhich slot that item would be equipped into (0-2, or -1 if not equippable)
durabilitythe dice that determine the item's initial durability
basePricethe total price, or the price per durability for items with durability, that the shop sells this item for
difficultythe relative power of this type of item (1-9)
onUsea function that implements what happens when this item is used or equipped (may be empty for unusable items, such as gold)
onUnequipa function that implements what happens when this item is unequipped (may be empty if not equippable)
